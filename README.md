# rn-skeleton-loading-view

Animated skeleton view that waits for content.

## Installation

```sh
npm install rn-skeleton-loading-view
```

## Usage

```js
import { multiply } from "rn-skeleton-loading-view";

// ...

const result = await multiply(3, 7);
```
